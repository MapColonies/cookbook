Special Phrases are common nouns or phrases that can be used in a search to identify a class of places. <br />
Those are keywords that cause Nominatim to search for a specific key=value pair present in the OSM entity.

## Examples

``` xml title='Request in English'
<VECTOR-NOMINATIM_URL>/search.php?format=jsonv2'&
q=school in tel aviv
```

``` xml title='Request in Hebrew'
<VECTOR-NOMINATIM_URL>/search.php?format=jsonv2'&
q=בית ספר בתוך תל אביב
```
