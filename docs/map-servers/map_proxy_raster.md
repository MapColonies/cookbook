## Overview
MapProxy is a tile server (WMS-C, TMS, WMTS, KML SuperOverlays). It reads data from:
- WMS sources (1.0.0–1.3.0)
- WMTS/TMS sources
- Mapserver and Mapnik configurations
- etc..
