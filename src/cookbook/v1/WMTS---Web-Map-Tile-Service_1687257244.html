Operations - See source

Operation

Description

GetCapabilities

Retrieves metadata about the service, including supported operations and parameters, and a list of the available layers.

GetTile

Retrieves a map image for a specified area and content.

GetFeatureInfo (optional)

Retrieves the underlying data, including geometry and attribute values, for a pixel location on a map.

GetCapabilities

Request

A WMTS server responding to a GetCapabilities request returns metadata about the service, including supported operations and parameters, and a list of the available layers.

An example of a GetCapabilities request is:

https://ows.terrestris.de/osm/service?
REQUEST=GetCapabilities&
SERVICE=WMTS

This is a link to a GetCapabilities request - demo.

GetTile

Request

http://cite.deegree.org/deegree-webservices-3.4-RC3/services/wmts100?
service=WMTS&
request=GetTile&
version=1.0.0&
Layer=cite&
Style=default&
Format=image/png&
TileMatrixSet=InspireCrs84Quad&
TileMatrix=17&
TileRow=4&
TileCol=4

Response
