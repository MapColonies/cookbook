Mapcolonies : WMTS Usage Examples

Created by Shlomi Koncha, last modified on Feb 16, 2022

How to consume WMTS ?

http://localhost:8080/wmts/Ortho_25cm/{TileMatrixSet}/{TileMatrix}/{TileCol}/{TileRow}.png

TileMatrixSet - define the grid name of the layer,

How to get Grid name? Grid name (TileMatrixSet) can be reached by GetCapabilities response:

below response shows:

we can see from above response the “TileMatrixSet” element that define the supported Grid names of the layer.

<TileMatrixSetLink>
<TileMatrixSet>MyGridName</TileMatrixSet>
</TileMatrixSetLink>

TileMatrix - define the zoom level. (Z)

TileCol - define the column (X)

TileRow - define the row (Y)

Ortho_25cm - the name of the layer

request example:

http://localhost:8080/wmts/bluemarble-1.0-OrthophotoHistory/newGrids/2/0/2.png

response:

Do not change the “/{TileMatrixSet}/{TileMatrix}/{TileCol}/{TileRow}.png” with custom values while consuming in applications - see examples below.

MapProxy automatically relates to those keys and parse the values “on the fly”.
